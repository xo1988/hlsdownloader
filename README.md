# hls downloader
> 一款方便查看下载hls视频的插件

# 安装
* [chrome web store](https://chrome.google.com/webstore/detail/hls-downloader-pro/mdfhlmcipgjjilidofjpkookbfiakfdj)

# 功能
* 支持自己手动添加hls格式m3u8链接
* 支持监控浏览网页内的hls格式m3u8链接自动添加到待下载列表中
* 支持点击查看视频内容
* 下载点播内容到本地
